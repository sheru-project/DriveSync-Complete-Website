🚀 Google Drive Clone – File Management Web App

A modern **cloud-based file management system** inspired by Google Drive, built using **PHP and MySQL** with a clean and professional UI/UX.

This application allows users to securely upload, manage, organize, and share files in a simple and efficient way.

✨ Features

* 🔐 Secure User Authentication (Login / Register)
* 📁 File Upload, Delete & Management
* 📂 Folder Organization & Search
* 🔗 File Sharing & Download
* 📊 Storage Dashboard
* 🎨 Responsive & Modern UI
* ⚡ Fast and Optimized Performance


🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
